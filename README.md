# Proyecto_Mysafeshop
Proyecto Final

📌 Nombre

MySafeShop (MSS)

🎯 Objetivo

Desarrollar una aplicación web de comercio electrónico enfocada en la venta de ropa de segunda mano de una tienda local, con el fin de reducir el impacto ambiental del fast fashion y promover el consumo responsable.

🌍 Enfoque diferencial
❌ NO marketplace
✅ UNA sola tienda
♻️ Impacto ambiental
💰 Precios accesibles
🧥 Ropa reutilizada
